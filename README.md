# 神马视频转换 (Shenma Video Converter)

这是一个基于 FFmpeg 的 Windows 图形化视频转码工具，支持多任务处理、批量转码、视频增稳、剪切、旋转等功能。

## 功能特性

- **多文件管理**：支持拖拽导入、批量删除。
- **灵活配置**：支持 MP4/MKV 格式，多种压缩质量（无损/高清/平衡/小体积），视频旋转，剪切。
- **高级功能**：集成 `vidstab` 视频增稳功能。
- **多任务并行**：支持多线程并行转码，可配置并发数。
- **实时进度**：直观的任务进度和状态监控。

## 环境要求

1. **Windows 操作系统**
2. **Python 3.8+**
3. **FFmpeg**: 必须安装 FFmpeg 并将其 `bin` 目录添加到系统环境变量 `PATH` 中，或者将 `ffmpeg.exe` 和 `ffprobe.exe` 放置在程序根目录下。

## 安装与运行

1. 安装 Python 依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：
   ```bash
   python main.py
   ```

## 注意事项

- **视频增稳**：启用增稳处理会显著增加转码时间，因为需要进行两遍处理（分析+转码）。
- **剪切功能**：输入的秒数支持小数（如 5.5）。
- **输出目录**：默认为源文件同级目录，可自定义。

## 开发说明

- `main.py`: 程序入口及业务逻辑。
- `gui.py`: 界面布局与控件定义 (PySide6)。
- `worker.py`: 多线程任务调度与 FFmpeg 交互核心逻辑。
